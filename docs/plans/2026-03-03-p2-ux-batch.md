3. Add right-click support: attach a `GestureClick` (button=3) to each FlowBoxChild that shows a simple context menu with "Play" and "Add to Queue":

   In `__init__`, add a `_current_menu` attribute for menu lifecycle.

   Add `_attach_mix_context_gesture(child)` method:
   ```python
   def _attach_mix_context_gesture(self, child):
       gesture = Gtk.GestureClick(button=3)
       def _on_right_click(_g, _n, x, y):
           tidal_id = getattr(child, "_mix_tidal_id", None)
           name = getattr(child, "_mix_name", None)
           if tidal_id and name and self._on_play_mix:
               menu = Gio.Menu()
               section = Gio.Menu()
               section.append("Play", "mix.play")
               menu.append_section(None, section)
               group = Gio.SimpleActionGroup()
               action = Gio.SimpleAction.new("play", None)
               action.connect("activate", lambda *_: self._on_play_mix(tidal_id, name))
               group.add_action(action)
               child.insert_action_group("mix", group)
               popover = Gtk.PopoverMenu.new_from_model(menu)
               popover.set_parent(child)
               popover.set_pointing_to(Gdk.Rectangle(int(x), int(y), 1, 1))
               popover.add_css_class("context-menu")
               popover.popup()
       gesture.connect("released", _on_right_click)
       child.add_controller(gesture)
   ```

   Call `self._attach_mix_context_gesture(child)` in `_populate_mixes()` and `_populate_playlists()` after creating each card.

4. Add required imports: `from gi.repository import Gdk, Gio` (Gdk and Gio if not already imported).

## Task 5: Fix Collection Artist Track Counts

**Problem:** In `_rebuild_merged_collections()`, Tidal artists get `track_count: 0` because the count loop only iterates tracks where `source == "local"`. Local artists show only local-favorited track counts (often 1).

**Files:**
- Modify: `auxen/views/collection.py`

**Fix:** Count ALL favorites for each artist (both local and Tidal), not just local ones. Also count Tidal tracks for Tidal-sourced artists.

**Steps:**

1. In `_rebuild_merged_collections()` (~line 1152-1176), change the artist counting loop to iterate ALL favorites (not just local):

   ```python
   # -- Artists --
   artists: dict[str, dict] = {}
   for ta in self._cached_tidal_artists:
       name = ta.get("name", "") or "Unknown Artist"
       artists[name] = {
           "artist": name,
           "track_count": 0,
           "sources": ["tidal"],
           "image_url": ta.get("image_url"),
           "tidal_id": ta.get("tidal_id"),
       }
   # Count ALL favorited tracks per artist (both local and tidal)
   for track in self._all_favorites:
       name = track.get("artist", "") or "Unknown Artist"
       source = track.get("source", "local")
       if name not in artists:
           artists[name] = {
               "artist": name,
               "track_count": 0,
               "sources": [source],
           }
       else:
           if source not in artists[name]["sources"]:
               artists[name]["sources"].append(source)
       artists[name]["track_count"] += 1
   self._merged_artists = list(artists.values())
   ```

**Verification:** Collection -> Artists tab -> track counts match actual favorited track counts for both local and Tidal artists.
